## Getting Help

### GitHub Issues

For bugs and feature requests, please use GitHub Issues:

- **Bug Reports**: [Create a bug report](https://github.com/kase1111-hash/Game-Wallet/issues/new?template=bug_report.md)
- **Feature Requests**: [Request a feature](https://github.com/kase1111-hash/Game-Wallet/issues/new?template=feature_request.md)

Before creating an issue:
1. Search existing issues to avoid duplicates
2. Use the provided issue templates
3. Include as much detail as possible

### Discussions

For questions, ideas, and general discussion:
- Use [GitHub Discussions](https://github.com/kase1111-hash/Game-Wallet/discussions) (if enabled)

## Issue Guidelines

When reporting issues, please include:

### For Bug Reports

- GLWM SDK version (`npm list @glwm/sdk`)
- Node.js version (`node --version`)
- Operating system and version
- Wallet provider being used
- Blockchain/chain information
- Minimal code to reproduce the issue
- Full error messages and stack traces

### For Feature Requests

- Clear description of the feature
- Use case and problem it solves
- Proposed API design (if applicable)
- Any alternatives considered
